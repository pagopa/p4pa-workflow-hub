### Prerequisites

Ensure the following tools are installed on your machine:

1. **Java 21+**
2. **Gradle** (or use the Gradle wrapper included in the repository)
3. **Docker** (for Helm-related tasks, optional)

### Building & Run

#### Build

```sh
./gradlew clean build
```

#### Run local

```sh
./gradlew bootRun
```

#### Write Locks

```sh
./gradlew dependencies --write-locks
```
